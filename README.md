# Smart-Reimburse
A smart Odoo module for managing employee expense claims and reimbursements with approval workflows, policy checks, and reporting.
Odoo Reimbursement Management

📌 Overview

This project is an Odoo module built during a hackathon to simplify and automate employee expense reimbursement. It provides a structured workflow for submitting claims, validating policies, approving requests, and tracking reimbursements.

✨ Features

Expense Submission: Employees can upload receipts and submit claims.

Approval Workflow: Multi-level approvals (Employee → Manager → Finance).

Policy Validation: Automatic checks against company reimbursement rules and limits.

Tracking & Reporting: Real-time status updates and department-wise expense reports.

Integration Ready: Works with Odoo’s accounting and payroll modules.

Demo Data: Sample employees and expenses for quick testing.

🛠️ Installation

Clone the repository:

git clone https://github.com/KartikiGawade/odoo-reimbursement-management.git

Place the module inside your Odoo addons directory.

Update Odoo app list and install Reimbursement Management.

🚀 Usage

Navigate to Expenses → Reimbursement Management in Odoo.

Submit a new claim with details and receipt.

Track approval status and reimbursement progress.

Generate reports for finance teams.
